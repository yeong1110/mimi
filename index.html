CSS란 무엇일까요?

종속형 시트 또는 캐스케이딩 스타일 시트는 마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어로, HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다. W3C의 표준이고, 레이아웃과 스타일을 정의할 때의 자유도가 높다. 기본 파일명은 style.css이다.

CSS Framework란 무엇일까요?

개발자와 웹 디자이너를 위해 바로 사용할 수 있는 CSS 라이브러리로 구성됩니다. 스타일시트는 탐색 모음, 글꼴, 색상 및 레이아웃 설정과 같은 웹 디자인의 일상적인 작업을 위해 준비됨. 모든 프로젝트를 처음부터 시작하는 대신 사용자 인터페이스를 만드는 도구를 제공하여 UI 개발자의 작업을 단순화

즉, 웹사이트를 더 빠르게 만드는 데 도움
